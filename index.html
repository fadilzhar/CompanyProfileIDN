    Program - Program IDN

    Program SMK RPL (Rekayasa Perangkat Lunak)

    Program 3 Tahun (2 tahun belajar + 1 tahun PKL)
    Memprogramm web dan mengembangkan
    Membuat aplikasi android

    Contoh Pembelajaran:
    1. Belajar HTML Dasar > Lihat Video
    2. Belajar CSS Dasar > Lihat Video
    3. Belajar Membuat Web dengan HTML & CSS > Lihat Video

    Program SMK TKJ (Teknik Komputer dan Jaringan)

    Program 3 Tahun (2 Tahun belajar + 1 Tahun PKL)
    Mempelajari Jaringan Komputer (Cisco & Mikrotik)
    Mempelajari Cloud Server

    Contoh Pembelajaran:
    1. Belajar Cisco > Lihat Video
    2. Belajar Mikrotik > Lihat Video
    3. Belajar Linux Server > Lihat Video

    Program SMK DKV (Design Komunikasi Visual)

    Program 3 Tahun (2 Tahun belajar + 1 tahun PKL)
    Mempelajari UI/UX Design
    Mempelajari Design Grafis
    Mempelajari Multimedia

    Contoh Pembelajaran:
    1. Membuat aplikasi mobile dengan Figma > Lihat Video
    2. Membuat aplikasi web dengan Figma Lihat Video
